# Writing as spaced repetition for understanding

Spaced repetition strengthens memory by revisiting material at increasing
intervals. But the same principle applies to understanding, not just recall.

When I revisit a note to connect it to a new idea, I'm forced to re-engage
with the original thought. Each revisit deepens my understanding because I
bring new context each time.

This is what Luhmann's slip-box achieves naturally: the act of filing a new
note requires reviewing related existing notes, creating an organic spaced
repetition cycle for ideas rather than facts.

The key insight is that understanding is strengthened by *rewriting* at
intervals, not just *rereading*. Passive review decays; active reformulation
compounds.

Connects to: active recall, elaborative encoding, Zettelkasten method
Inspired by: "How to Take Smart Notes" (Ahrens)
