## Personal Knowledge — domain triggers

### When to search the wiki

- Before writing about a topic → check what you already know
- "What have I read about X?" → search for related notes
- Making connections between ideas → search broadly for related concepts
- Revisiting an old idea → search to find the original note and its connections
- Processing a new source → search for notes that might connect to it

### Domain-specific queries

- Find related ideas: `wiki_search("spaced repetition")`
- Trace inspiration: `wiki_ontology_query` with relation "inspired_by"
- Find developments: `wiki_ontology_query` with relation "develops"
- Find contrasts: `wiki_ontology_query` with relation "contrasts_with"
- Browse by type: `wiki_list` with entity type "permanent_note" or "fleeting_note"

### What to capture

- Quick idea or observation → `wiki_learn` type "fleeting" (to be refined later)
- Refined insight in your own words → `wiki_learn` type "permanent"
- Connection between two ideas → `wiki_learn` type "connection" referencing both
- Reading note from a source → `wiki_capture` from the text
- Question raised by a note → `wiki_learn` type "question" for future investigation
